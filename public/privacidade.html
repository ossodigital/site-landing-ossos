1. Informações Gerais

Esta Política de Privacidade descreve como o Tattoo até os Ossos coleta, usa e protege as informações pessoais de nossos clientes, em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

2. Dados Coletados

2.1 Dados de Identificação:

  • Nome completo
  • Data de nascimento
  • Documentos de identidade

2.2 Dados de Contato:

  • Telefone/WhatsApp
  • E-mail
  • Endereço

2.3 Dados de Saúde:

  • Alergias conhecidas
  • Condições médicas relevantes
  • Medicamentos em uso
  • Histórico de procedimentos anteriores

2.4 Dados de Mídia:

  • Fotografias dos trabalhos realizados
  • Imagens para fins de divulgação (com consentimento)

3. Finalidade do Tratamento

Utilizamos seus dados pessoais para:

  • Prestação dos serviços de tatuagem e piercing
  • Comunicação sobre agendamentos e procedimentos
  • Cuidados pós-procedimento
  • Arquivo histórico dos trabalhos realizados
  • Divulgação dos trabalhos (com consentimento expresso)
  • Cumprimento de obrigações legais

4. Base Legal

O tratamento dos dados pessoais é baseado em:

  • Consentimento: Para uso de imagens em divulgação
  • Execução de contrato: Para prestação dos serviços
  • Interesse legítimo: Para melhoria dos serviços
  • Proteção da vida: Para informações de saúde essenciais

5. Compartilhamento de Dados

Seus dados podem ser compartilhados apenas:

  • Com profissionais de saúde, quando necessário
  • Com autoridades competentes, quando exigido por lei
  • Com fornecedores de serviços (sob contrato de confidencialidade)

6. Retenção de Dados

  • Dados de saúde: 20 anos (conforme CFM)
  • Dados contratuais: 5 anos
  • Dados para divulgação: Enquanto houver consentimento

7. Seus Direitos

Você tem direito a:

  • Confirmação da existência de tratamento
  • Acesso aos dados
  • Correção de dados incompletos ou inexatos
  • Anonimização, bloqueio ou eliminação
  • Portabilidade dos dados
  • Eliminação dos dados tratados com consentimento
  • Revogação do consentimento

8. Segurança

Implementamos medidas técnicas e organizacionais para proteger seus dados:

  • Acesso restrito aos dados
  • Arquivos físicos em local seguro
  • Backup regular dos dados
  • Treinamento da equipe

9. Alterações na Política

Esta política pode ser alterada periodicamente. Alterações serão comunicadas através de nossos canais oficiais.
